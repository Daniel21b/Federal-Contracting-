Data Sources

This project uses federal contracting data from the following sources:

  • USA Spending.gov

    Official source of federal spending data for the U.S. Government

  • Federal Procurement Data System

    The primary source for U.S. government procurement data

  • State-specific procurement databases

    Additional data from Virginia, California, and Maryland procurement systems

Methodology

Our data collection and analysis process includes:

  1. Data extraction from multiple government sources
  2. Data cleaning and normalization across sources
  3. Aggregation by state, agency, and industry sector
  4. Trend analysis over selected time periods

About This Project

Project Background

This data visualization dashboard was created to provide transparency and insights into federal contracting patterns across Virginia, California, and Maryland. By visualizing this data, we aim to help stakeholders understand how federal dollars are distributed and identify patterns in government spending.
